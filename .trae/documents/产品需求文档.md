## 1. 产品概述

农机作业调度系统是一个面向农业生产管理的可视化调度平台，通过地图实时展示农机位置与作业状态，支持作业任务智能派单、轨迹回传与数据分析，帮助调度员高效管理农机资源、提升作业效率。

- 目标用户：农场调度员、农机管理员
- 核心价值：可视化调度、智能派单、作业数据量化分析

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|---------|---------|
| 调度员 | 账号登录 | 查看全部农机、创建任务、调度派单、查看统计数据 |
| 农机手 | 移动端登录 | 查看任务、接单、轨迹上报 |

### 2.2 功能模块

1. **地图调度主面板**：农机实时位置分布、地块展示、作业状态可视化
2. **作业任务管理**：创建任务（选地块+选机具+作业类型）、智能派单、任务列表
3. **农机管理**：农机列表、状态筛选、详情查看
4. **轨迹回放**：作业轨迹实时回传、历史轨迹回放
5. **数据统计**：按日/周统计作业面积、油耗、作业时长

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|----------|
| 调度主面板 | 地图视图 | Leaflet 地图展示所有农机GPS位置，不同状态用不同颜色标识，支持缩放平移 |
| 调度主面板 | 农机列表侧栏 | 左侧农机列表，按状态分类（作业中/空闲/维修/转场），点击定位 |
| 调度主面板 | 地块图层 | 地图上显示作业地块多边形，不同颜色区分作业类型 |
| 调度主面板 | 快捷统计 | 顶部概览卡片：在线农机数、作业中、今日作业面积、总油耗 |
| 任务创建面板 | 任务表单 | 选择地块、选择农机器具、设定作业类型（耕/种/管/收）、设定时间 |
| 任务创建面板 | 智能派单 | 系统自动匹配最近的空闲农机，支持手动调整 |
| 任务列表 | 任务管理 | 进行中/待分配/已完成任务列表，支持状态筛选 |
| 轨迹回放 | 轨迹播放器 | 选择农机和日期，播放作业轨迹，支持倍速、进度拖动 |
| 统计面板 | 数据图表 | 按日/周切换，展示每台农机作业面积柱状图、油耗趋势图 |

## 3. 核心流程

调度员创建作业任务 → 选择地块与作业类型 → 系统自动匹配最近空闲农机 → 推送任务给农机手 → 农机手确认接单 → 作业开始，轨迹实时回传 → 作业完成，数据统计入库

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深绿色系（#15803d）代表农业、生机，搭配土黄色（#ca8a04）作为强调色
- **辅助色**：状态色 - 作业中（绿色）、空闲（蓝色）、维修（红色）、转场（橙色）
- **整体风格**：现代工业感、数据可视化驱动、深色侧边栏 + 浅色内容区
- **布局**：左侧导航 + 右侧主内容区（地图占主要空间），顶部工具栏
- **字体**：无衬线现代字体，清晰易读
- **图标风格**：线性图标，简洁专业

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 调度主面板 | 顶部栏 | Logo、搜索框、状态筛选标签、用户头像 |
| 调度主面板 | 左侧农机列表 | 卡片式列表，状态标签色条，滚动容器 |
| 调度主面板 | 中央地图 | Leaflet 地图，自定义 Marker，地块多边形图层 |
| 调度主面板 | 右侧任务面板 | 折叠式面板，当前进行中任务列表 |
| 调度主面板 | 底部统计条 | 关键指标数据卡片，横向排列 |
| 任务创建 | 模态弹窗 | 分步表单，地图选点，农机选择器 |
| 统计面板 | 图表区 | 柱状图 + 折线图组合，日期切换器 |

### 4.3 响应性

- 桌面端优先设计，主视区 >= 1440px
- 平板端：侧边栏可折叠，地图自适应
- 移动端：简化为列表视图 + 全屏地图切换

### 4.4 地图可视化

- 底图：使用 OpenStreetMap 或简约风格瓦片
- 农机 Marker：自定义 SVG 图标，带方向指示，状态光晕动画
- 地块：半透明填充多边形，边框高亮
- 轨迹：渐变折线，起点终点标记
- 悬浮效果：农机信息弹窗，显示实时数据
